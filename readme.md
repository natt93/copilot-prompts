🧩 VISÃO DO DIA A DIA

Como é o trabalho de um(a) Desenvolvedor de Automação / IA:

Criar scripts e aplicações para automatizar tarefas repetitivas

Desenvolver soluções com IA (ex: classificação de dados, chatbots, análise)

Integrar APIs e sistemas (ex: serviços de IA na nuvem)

Analisar processos e identificar oportunidades de automação

Testar, corrigir bugs e otimizar desempenho das soluções

🧠 MAPA DE SKILLS
CORE SKILLS (essenciais):

Lógica de programação e estrutura de dados

Programação em C# e .NET

Fundamentos de IA (machine learning básico e APIs de IA)

NICE-TO-HAVE (complementares):

Conceitos de cloud (Azure principalmente)

Boas práticas (Clean Code, testes básicos)

FERRAMENTAS E TECNOLOGIAS:

C# / .NET

Azure AI / APIs de IA

Git e GitHub

📅 ROADMAP DE 90 DIAS

ADAPTADO PARA: 15 horas/semana

MÊS 1 - FUNDAMENTOS
SEMANA 1-2:

Revisar lógica de programação (condições, loops, funções)

Aprender base de C# (variáveis, classes, métodos)

SEMANA 3-4:

Programação orientada a objetos (POO) em C#

Criar pequenos projetos (ex: calculadora, automação simples)

MÊS 2 - PRÁTICA
SEMANA 5-6:

Introdução a APIs (consumir APIs com C#)

Criar automações simples (ex: ler arquivo, processar dados)

SEMANA 7-8:

Introdução à IA (uso de APIs de IA, sem matemática pesada)

Criar um mini projeto com IA (ex: classificar texto ou gerar respostas)

MÊS 3 - PORTFÓLIO E PREPARAÇÃO
SEMANA 9-10:

Iniciar projeto principal de portfólio

Versionar código no GitHub

SEMANA 11-12:

Refinar projeto (melhorar código, interface simples, documentação)

Preparação para entrevistas + revisão geral

🚀 PROJETO DE PORTFÓLIO
PROJETO: Assistente de Automação Inteligente
O QUE FAZER:

Criar uma aplicação em C# que automatize tarefas simples (como leitura de arquivos ou organização de dados) e utilize IA para interpretar ou responder informações.

Exemplo: um sistema que lê um arquivo e usa IA para resumir ou classificar o conteúdo.

ENTREGÁVEIS:

Código no GitHub

Aplicação funcional (console ou simples interface)

README explicando o projeto

CRITÉRIOS DE ACEITAÇÃO:

Código organizado e legível

Integração com API de IA funcionando

Demonstração clara da automação

DICA:

Comece simples: primeiro automatize sem IA… depois “injete inteligência” na solução. Isso evita travar no começo 🔑
